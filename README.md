# SentimentIQ — Product Sentiment Analyzer

Analyze customer reviews with AI-powered sentiment analysis.

## Tech Stack
- **Backend:** Flask, Python
- **ML:** scikit-learn (Logistic Regression + TF-IDF)
- **Database:** MongoDB
- **Frontend:** HTML/CSS, Chart.js, Font Awesome

## How It Works
1. User searches for a product name
2. App queries MongoDB for matching reviews
3. Reviews are vectorized using TF-IDF
4. Logistic Regression model predicts sentiment (positive/negative)
5. Results are displayed with charts and statistics

## Features
- Search any product and analyze review sentiment
- Interactive bar and doughnut charts
- Sentiment score with positive/negative breakdown
- Sample review cards with sentiment labels
